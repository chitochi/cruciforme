Cruciforme

A form endpoint for static websites

<form
	action="https://crucifor.me"
	method="post"
	enctype="multipart/form-data">

	<input name="lastName" type="text">
	<input name="firstName" type="text">

	<input
		name="cruciforme-mail"
		value="your@awesome.mail"
		type="hidden">
</form>

It's easy !

Take a form, for example:

<form>
	<input name="lastName" type="text">
	<input name="firstName" type="text">
</form>

Point the form to crucifor.me:

<form
	action="https://crucifor.me"
	method="post"
	enctype="multipart/form-data">

	<input name="lastName" type="text">
	<input name="firstName" type="text">
</form>

Set the email address where to send the answers:

<form
	action="https://crucifor.me"
	method="post"
	enctype="multipart/form-data">

	<input name="lastName" type="text">
	<input name="firstName" type="text">

	<input
		name="cruciforme-mail"
		value="your@awesome.mail"
		type="hidden">
</form>

That's it!

You can redirect the user to a custom page:

<input
	name="cruciforme-success"
	value="your.awesome.site/success"
	type="hidden">
<input
	name="cruciforme-error"
	value="your.awesome.site/error"
	type="hidden">

Finally, you can customize the subject of the email:

<input
	name="cruciforme-subject"
	value="Yay! An email!"
	type="hidden">
